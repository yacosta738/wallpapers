## Usage

### Clone the repository

```bash
git clone https://github.com/yourusername/wallpapers.git
```

### Setting as wallpaper

- **macOS**: System Settings → Desktop & Dock → Wallpaper → Add Folder
- **Windows**: Settings → Personalization → Background
- **Linux**: Use `feh`, `nitrogen`, or your DE's settings

### Live Wallpapers

Use with wallpaper engines like:

- [Wallpaper Engine](https://store.steampowered.com/app/431960/Wallpaper_Engine/) (Windows)
- [MPV](https://mpv.io/) with `--loop` (Linux/macOS)
- [Komorebi](https://github.com/cheesecakeufo/komorebi) (Linux)
